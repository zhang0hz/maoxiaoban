# V0.10 Settings / Product Polish

Implemented:

- Settings window height reduced to 500.
- Short tabs no longer use fixed-height scroll containers.
- Reminder tab keeps scrolling because it has editable rows.
- Overview tab:
  - open data folder
  - reset all settings
- Classification tab:
  - shows current app and bundle id
  - set current app as work, leisure, communication, neutral, or clear
  - open data folder
  - reset all settings

Reset all settings:

- resets mode, auto-position, pause, size, pinned position
- clears app classification overrides
- resets reminders to defaults
