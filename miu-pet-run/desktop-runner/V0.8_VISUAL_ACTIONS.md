# V0.8 Visual Actions

Goal: make Miu feel more alive using existing Phase 1 sprites.

Implemented:

- Short reaction action system:
  - temporary actions can override normal behavior for a few seconds
  - normal behavior resumes automatically
- Mouse reactions:
  - mouse enters Miu: `purr`
  - mouse leaves Miu: `purr`
- Drag reaction:
  - after dragging Miu: `stretch`
- Reminder reactions:
  - complete reminder: `celebrate`
  - snooze reminder: `comfort`
  - skip reminder: `groom`
  - close bubble: `peek`
- Settings reaction:
  - saving reminder settings: `celebrate`
- Behavior action stability:
  - less rapid action flicker
  - smoother visual personality

Notes:

- No new generated sprites in this pass.
- Uses existing frames: `purr`, `stretch`, `celebrate`, `comfort`, `groom`, `peek`.
- `sit-watch` is currently aliased to `purr` because its original frames had a gray-blue color shift.
- Future V0.8.x can add new sprite assets: blink, ear twitch, tail sway, yawn, tiny paw wave.
