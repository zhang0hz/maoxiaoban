# V0.5 Light Reminders

Goal: let Miu give gentle local reminders without becoming noisy or blocking work.

Implemented in V0.5.0:

- Local reminder scheduler inside `MiuDesktopRunner.swift`.
- Reminder state persisted at:

```text
~/Library/Application Support/Miu/reminders.json
```

- Default reminder set:
  - `喝水`: every 90 minutes, active 09:00-22:00, action `purr`.
  - `休息`: every 60 minutes, active 09:00-22:30, action `stretch`.
  - `久坐`: every 90 minutes, active 09:00-22:00, action `stretch`.
  - `睡觉`: fixed 23:30, active 22:30-23:59, action `sleep`.
- Interval reminders seed `lastCompletedAt` on first launch, so Miu does not immediately show every default reminder.
- Reminders do not fire when a fullscreen/presentation-sized window is detected.
- Reminders do not fire during quiet work states.
- Non-sleep reminders do not fire in night/sleepy modes.

Implemented in V0.5.1-lite:

- Reminder bubble above Miu.
- Bubble actions:
  - `完成`: mark current reminder complete and reset interval.
  - `稍后10`: snooze current reminder 10 minutes.
  - `跳过`: skip the reminder for today.
  - `关闭`: close current bubble without marking complete.
- Menu bar / right-click menu has `提醒` submenu with current reminder actions or next reminder summary.
- Settings window has `提醒气泡自动消失`:
  - `10秒`
  - `30秒`
  - `60秒`
  - `不自动`
- Settings window can edit each built-in reminder:
  - enable / disable
  - interval minutes for water, rest, and stand reminders
  - fixed `HH:mm` time for sleep reminder
  - active start / end time range
  - reminder message text
  - save edited reminders
  - restore default reminders

Implemented in V0.5.2:

- Custom reminder rows:
  - `新增自定义提醒` creates an interval reminder with default 60-minute cadence.
  - `删除` removes a reminder row.
  - `恢复默认提醒` restores built-in defaults.
- Reminder menu has richer active-reminder actions:
  - `稍后 10 分钟`
  - `稍后 30 分钟`
  - `稍后 60 分钟`
- Reminder menu shows today's completed/skipped count.
- Reminder bubble also offers `稍后30`.

Config shape:

```json
{
  "bubbleAutoDismissSeconds": 10,
  "reminders": [
    {
      "id": "water",
      "kind": "water",
      "enabled": true,
      "intervalMinutes": 90,
      "fixedTime": null,
      "activeStart": "09:00",
      "activeEnd": "22:00",
      "message": "喝点水吧",
      "action": "purr"
    }
  ]
}
```

Notes:

- `bubbleAutoDismissSeconds: null` means no auto-dismiss.
- Time fields use `HH:mm`; invalid values fall back to the previous saved value.
- Interval reminders are clamped to 5-720 minutes.
- Future reminder work can add one-shot calendar reminders and smarter suppression during calls.
