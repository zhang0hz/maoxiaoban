# V0.8.1 Visual Asset Rework

Goal: keep all visual reactions identity-safe.

Implemented:

- Added deterministic color audit script:

```bash
miu-pet-run/desktop-runner/color_audit.py \
  --frames-root miu-pet-run/phase1-actions/frames
```

- Audit checks all action folders against `loaf` as color reference.
- `sit-watch` remains protected:
  - frames replaced with color-safe `purr` frames
  - runtime maps `sit-watch` to `purr`
- App bundle frames are audited after packaging.

Design decision:

- No new generated sprites in this pass.
- Priority is preserving Miu identity and preventing color drift.
- Future asset generation should use the same color audit before install.
