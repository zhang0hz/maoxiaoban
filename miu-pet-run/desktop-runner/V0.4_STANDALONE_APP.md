# V0.4 Standalone App

Goal: make Miu run outside Codex/project folders.

## Done

- `Miu.app` app bundle.
- Action frames bundled inside:

```text
Miu.app/Contents/Resources/frames
```

- Runtime auto-loads bundled frames when launched as app.
- App classification config stored outside project:

```text
~/Library/Application Support/Miu/app-classifications.json
```

- Native settings window:
  - mode: 自动 / 工作 / 休闲 / 睡觉
  - 自动贴边
  - 暂停动画
  - 登录时自动启动 Miu
  - size: 小 / 中 / 大
  - current detected status/reason

- Settings persist at:

```text
~/Library/Application Support/Miu/settings.json
```

- Distribution output:

```text
miu-pet-run/dist/Miu.app
miu-pet-run/dist/Miu.zip
```

- Ad-hoc codesign attempted when `codesign` exists.

## Build

```bash
miu-pet-run/desktop-runner/build-dist.sh
```

## Run Standalone Build

```bash
open miu-pet-run/dist/Miu.app
```

## Install To Applications

```bash
miu-pet-run/desktop-runner/install-to-applications.sh
```

## Still Missing
