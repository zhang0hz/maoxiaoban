# V0.6 Settings UI

Goal: make Miu settings easier to scan and edit.

Implemented:

- Settings window now uses tabs:
  - `概览`
  - `行为`
  - `提醒`
  - `分类`
- `概览` shows current detected state, reason, next reminder, today reminder summary, and data path.
- `行为` contains mode, auto-position, pause, login-at-startup, and size controls.
- `提醒` contains bubble auto-dismiss and the full reminder editor.
- `分类` shows current app and classification config path.

Notes:

- App classification editing still lives in the menu bar / right-click menu.
- Future V0.6.x can move classification editing into this tab.
