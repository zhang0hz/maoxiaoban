# V1.0.4 Icon and Release Polish

Goal: make 猫小伴 look like a releasable product while preserving the desktop pet identity.

Implemented:

- Added product app icon generation from the canonical blue bi-color ragdoll cat asset.
- Icon design:
  - macOS-style rounded square
  - light blue product background
  - subtle border and depth
  - full cat face/body identity preserved
  - no text in the icon
- Added generated assets:
  - `Assets/app-icon-1024.png`
  - `Assets/AppIcon.iconset`
  - `Assets/AppIcon.icns`
- Added build scripts:
  - `make-app-icon.py`
  - `build-icon.sh`
- `build-app.sh` now builds and bundles `AppIcon.icns`.
- `Info.plist` now declares `CFBundleIconFile`.

Notes:

- The icon is generated locally from existing project art, not from a new AI image, to avoid identity drift.
- ICNS is written directly as standard PNG-backed ICNS chunks because `iconutil` rejected the generated iconset on this machine.
