# V0.12 Classification + Reminder Smartness

Implemented:

- Classification tab now shows saved app classification overrides.
- Current app can be classified directly in Settings:
  - work
  - leisure
  - communication
  - neutral
  - clear
- Overview tab shows deferred reminder status.
- Reminders due during quiet work, meetings, fullscreen, or large media windows remain deferred instead of being marked complete or skipped.
- Deferred reminders surface once quiet state clears.

Notes:

- Suppression uses existing system signals: front app, window coverage, idle time, day mode.
- Future V0.12.x can add per-app batch edit/delete UI beyond the read-only saved override list.
