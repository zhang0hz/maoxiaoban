# V0.20 Visual Motion Enhancement

Goal: improve life-like motion without making the pet noisy.

Implemented changes:

- Added a low-frequency micro-expression selector for calm states.
- `blink` and `ear-twitch` are treated as brief companion variations rather than primary states.
- `tail-sway` acts as the breathing-like leisure and idle loop before any walking starts.
- Micro-expressions now wait at least 48 seconds and rotate lightly between blink and ear twitch.
- Walking continues to use direction-specific `walk-right` and `walk-left` frames.
- State transitions choose existing formal actions instead of hard switching directly.

Visual QA:

- Watch active work mode for at least 5 minutes; micro-expressions should be rare.
- Watch leisure mode for at least 10 minutes; tail motion should appear before walking.
- Confirm no moonwalk: right-edge travel uses `walk-right`, left-edge travel uses `walk-left`.
- Confirm sleep and sleepy transitions use `groom` or `yawn` before settling.
