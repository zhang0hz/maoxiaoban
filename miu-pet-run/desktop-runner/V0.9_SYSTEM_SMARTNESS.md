# V0.9 System Smartness

Goal: reduce interruptions by understanding more system states.

Implemented:

- Meeting / communication:
  - Zoom, Teams, Slack, WeChat, FaceTime, Messages, Discord
  - treated as quiet work
  - reminders suppressed
- Presentation / fullscreen:
  - PowerPoint, Keynote, or near-fullscreen coverage
  - treated as quiet, low-distraction state
- Deep work:
  - work app with large focused window
  - reminders suppressed
  - Miu stays calmer
- Media / leisure:
  - Music, TV, QuickTime, IINA, VLC, Spotify, Steam, media apps
  - large media window suppresses reminders
  - Miu stays low-distraction
- Browser:
  - work hours or large browser window => work
  - large focused browser window => quieter behavior
  - non-work hours => leisure

Settings UI adjustment:

- Settings window height reduced from 680 to 560.
- Tab contents now scroll, so lower blank space is reduced without losing reminder rows.
