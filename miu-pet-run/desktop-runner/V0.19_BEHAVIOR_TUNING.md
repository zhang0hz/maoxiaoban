# V0.19 Behavior Tuning

Goal: make 猫小伴 calmer during work and less jumpy between behavior states.

Implemented changes:

- Work behavior now favors `loaf` as the default companion action.
- Work state action minimum duration increased to 9 seconds.
- Leisure walking waits until the user has been idle for more than 8 minutes.
- Leisure `peek` waits until idle time passes 3 minutes.
- Idle grooming/stretching thresholds are less eager.
- Reminder display uses a tiny `paw-wave` / `blink` / `yawn` lead-in when possible.
- State transitions use gentle bridge actions such as `wake`, `groom`, `peek`, or `loaf`.

Manual QA:

- Work for 30 minutes in an IDE or browser and confirm action changes feel rare.
- Enter leisure time or force leisure mode and confirm walking does not start immediately.
- Trigger a reminder and confirm the action before the bubble is subtle rather than abrupt.
- Force morning, sleepy, and sleep modes from the menu and confirm transitions do not jump sharply.
