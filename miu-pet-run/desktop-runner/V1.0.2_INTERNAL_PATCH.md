# V1.0.2 Internal Patch Scope

Goal: keep the current internal build stable while later behavior, visual, settings, and distribution work continues locally.

Included locally:

- Fix P1 click interception: hidden/fullscreen pet windows now disable mouse events, and pet windows covered by foreground windows do not keep capturing clicks.
- Keep quiet work states low-distraction without disabling direct pet drag, click, or context-menu interaction.
- Improve multi-display front-window coordinate handling by scoring raw, per-screen converted, and desktop converted candidates against the active screen.
- Cache `PetView` alpha hit testing so mouse events do not rebuild bitmap data on every hit test.
- Clamp manual pet dragging and reminder bubble placement to the current visible screen.
- Keep transparent sprite hit testing from V1.0.1.
- Keep menu truncation and tooltip support for long disabled rows.
- Keep multi-window avoidance scoring.
- Keep release install behavior that syncs `/Applications/猫小伴.app` after building dist.
- Include V1.0.2 hotfix checks and roadmap static checks in `smoke-test.sh` so later P1-P4 work is verified together.

Boundaries:

- No Developer ID signing or notarization in this phase.
- No force push or GitHub upload while network is unstable.
- No destructive cleanup of old visual assets; old material is marked through asset manifests first.

Manual QA:

- Launch from `/Applications/猫小伴.app` and confirm it opens the latest local build.
- Put another App over the pet and confirm foreground buttons, inputs, lists, and drag targets receive clicks normally.
- Enter fullscreen or near-fullscreen work and confirm the hidden pet does not intercept clicks.
- Drag the pet toward all screen edges and confirm it remains recoverable inside the visible screen.
- Trigger a reminder near top/side screen edges and confirm the bubble remains fully clickable.
- Open settings and confirm the overview shows version, build, Bundle ID, and internal distribution status.
- Trigger or inspect reminders and confirm invalid times are blocked with inline text.
- Work for 30 minutes and confirm the pet remains quiet while drag, click, and right-click still work when the pet is visible.
- Switch to leisure or leave the Mac idle and confirm walking starts only after a long idle period.
- Record results in `V1.0.2_QA_RECORD.md`.
