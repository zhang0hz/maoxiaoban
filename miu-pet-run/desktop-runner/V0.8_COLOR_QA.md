# V0.8 Color QA

Goal: prevent action changes from changing Miu identity or color.

Audit reference:

- Reference action: `loaf`
- Check method: non-transparent pixel RGB average, plus blue-bias check.
- Bug found: `sit-watch` was a different gray-blue pixel cat.

Fix:

- Replaced `sit-watch` frames with color-safe `purr` frames.
- Runtime action resolver maps `sit-watch` to `purr` as an extra guard.
- Mouse-leave reaction now uses `purr`.
- Work / idle active-watch behavior now uses `purr` instead of `sit-watch`.
