# V0.7 Behavior Intelligence

Goal: make Miu choose calmer, more context-aware actions.

Implemented:

- Work mode action chooser:
  - meetings / communication apps: `loaf`
  - large focused window: `sleep`
  - quiet work: `loaf`, then occasional `groom` after idle time
  - active work without large window: `sit-watch`
  - long idle in work mode: `stretch`
- Leisure mode action chooser:
  - large media window: `loaf`
  - long idle: edge walk
  - medium idle: `peek`
  - active leisure: `purr`
- Idle mode action chooser:
  - long idle: `sleep`
  - medium idle: `stretch` / `groom`
  - active idle: `sit-watch`
- Action stability guard:
  - behavior actions hold for a short minimum time
  - reduces jitter between frames/actions

Notes:

- Reminder suppression still respects fullscreen/presentation-sized windows and quiet work.
- Communication apps already mark quiet work, so reminders avoid meetings.
