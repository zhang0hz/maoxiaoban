# V0.4.1 Settings Persistence + Launch At Login

## Done

- Settings persist locally:

```text
~/Library/Application Support/Miu/settings.json
```

- Persisted fields:
  - manual mode: 自动 / 工作 / 休闲 / 睡觉
  - 自动贴边
  - 暂停动画
  - size scale: 小 / 中 / 大
  - manual position pin
  - window x/y

- Settings load on app launch.
- Dragged position saves on mouse release.
- Settings window has:

```text
登录时自动启动 Miu
```

- Login item uses `SMAppService.mainApp`.

## Notes

- Login item requires installed app bundle to be stable. Use `/Applications/Miu.app`.
- macOS may show login-item permission state in System Settings.
- App classification overrides remain separate:

```text
~/Library/Application Support/Miu/app-classifications.json
```
