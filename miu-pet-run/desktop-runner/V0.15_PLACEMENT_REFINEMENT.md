# V0.15 Placement Refinement

Goal: make safe placement steadier and user-controllable.

Implemented:

- Placement preference in Settings > Behavior:
  - Auto
  - Lower right
  - Lower left
  - Upper right
  - Upper left
- Auto-position now stays on the display where Miu currently is.
- Auto mode has placement hysteresis:
  - if Miu is already inside the visible screen and not overlapping the active window, it stays put
  - this reduces jumpiness from repeated behavior updates
- Fixed placement preferences still avoid the active window when the preferred corner would overlap it.
- Size changes and reset position reuse the same safe-placement logic.

Persistence:

- `placementPreference` is saved in:

```text
~/Library/Application Support/Miu/settings.json
```

Notes:

- User-pinned manual position still overrides auto placement.
- Leisure walking still follows the visible lower edge of the current display.
